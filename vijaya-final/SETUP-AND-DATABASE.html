▶️ How to Run the Software
1
Easiest Method — Just double-click
Unzip the downloaded file → Open folder vijaya-final → Double-click login.html → Opens in Chrome
✅ This works on Windows, Mac, Linux. No installation needed.
2
If blank page — Run Python Server (Windows)
Open Command Prompt (press Win+R → type cmd → Enter)
Step 1 — Go to the folder
cd C:\Users\YourName\Downloads\vijaya-final
Step 2 — Start server
python -m http.server 8080
Step 3 — Open in Chrome
http://localhost:8080/login.html
3
If Python not installed — Use Node.js
cd C:\Users\YourName\Downloads\vijaya-final npx serve .
Then open the URL it shows (usually http://localhost:3000)
4
VS Code — Live Server (Easiest for developers)
1. Open VS Code → File → Open Folder → select vijaya-final
2. Install extension: Live Server by Ritwick Dey
3. Right-click login.html → Open with Live Server

🗄️ Where Database is Stored
📍
Physical location on your computer
Windows — Chrome
C:\Users\YourName\AppData\Local\Google\Chrome\User Data\Default\IndexedDB\
Windows — Edge
C:\Users\YourName\AppData\Local\Microsoft\Edge\User Data\Default\IndexedDB\
Mac — Chrome
~/Library/Application Support/Google/Chrome/Default/IndexedDB/
⚠️ Never delete this folder. Do NOT clear Chrome browsing data. Always use the same browser on the same computer.
👁️ How to View Database in Chrome — Step by Step
1
Open Software in Chrome
Open login.html in Google Chrome and log in
2
Open Developer Tools
Press F12 on keyboard — OR — Right-click anywhere → Inspect
3
Click Application Tab
In DevTools, click the Application tab at the top. If you don't see it, click >> to show more tabs
4
Navigate to IndexedDB
In the left panel: Storage → IndexedDB → VijayaMedicalsPMS
5
View Any Table
Click any table name to see all records:
medicines → all medicines
bills → all bills
customers → all customers
suppliers → all suppliers
settings → shop settings
6
Click a Record to See Full Details
Click any row in the table → the full record shows in the right panel with all field values
✅ You can see all data live. Any changes made in the software show here immediately. Use this to verify your data is saved correctly.

💾 Backup & Restore Data
📥
Download Backup (in software)
Go to Downloads & PDF page → Click Download JSON Backup → Saves as vijaya-medicals-backup-DATE.json
✅ Do this every week. Store the .json file on USB drive or Google Drive.
📤
How to restore if data is lost
Currently data restore requires re-entering from the backup JSON file. The JSON file contains all your data in readable format — open it in Notepad to view all records.
⚠️ If Chrome data is cleared, the backup JSON is the only way to recover. Keep it safe!

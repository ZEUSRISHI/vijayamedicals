ℹ️ Best setting for billing: Use 10 CPI (Courier 10pt) for 136 columns. This fits all bill columns perfectly on 15" continuous feed paper.
1
Install Epson Driver (Windows)
1
Go to epson.com → Support → Search LQ-2090II
2
Download Epson LQ-2090II ESC/P Driver for your Windows version (32/64-bit)
3
Run installer → Follow prompts → Connect printer via USB cable when asked
4
After install: Open Control Panel → Devices & Printers — verify EPSON LQ-2090II appears
5
Right-click → Set as Default Printer
⚠️ Important: Load continuous-feed paper into the tractor feed BEFORE installing driver, so Windows detects correct paper size.
2
Windows Printing Preferences
1
Control Panel → Devices & Printers → Right-click EPSON LQ-2090II → Printing Preferences
2
Paper/Quality tab:
Paper Size: Custom — 13.6in × 11in
Orientation: Portrait
3
Advanced tab:
Print Quality: NLQ (Near Letter Quality)
Character Pitch: 10 CPI
Font: Courier
4
Page Setup:
Top margin: 0.2 inch
Left margin: 0.3 inch
Right margin: 0.3 inch
✅ Click Apply → OK to save. These settings apply every time you print.
3
Chrome Browser Print Settings
1
When bill print window opens, Chrome print dialog appears automatically
2
Destination: Select EPSON LQ-2090II (not "Save as PDF")
3
More settings → Paper size: Custom — 13.6in × 11in
4
Margins: Minimum or None
5
Scale: 100% — never scale up/down
6
Headers and footers: OFF (uncheck)
7
Background graphics: OFF
8
Click Print — the bill sends to Epson LQ-2090II
⚠️ First time only: Do a test print on plain paper first. Check if all 136 columns fit within the paper width. Adjust left margin if needed.
4
Loading Paper — Tractor Feed
1
Use 15 inch wide continuous-feed paper (tractor holes on both sides)
2
Open the tractor feed cover on the back of the printer
3
Align the left tractor to position 0. Set right tractor to match paper width (15 inch = position 15)
4
Feed the paper through, align holes onto tractor pins on both sides
5
Press Load/Eject button — printer advances paper to top-of-form
6
Set the paper selector lever to Tractor position
✅ Recommended paper: 15" × 11" single-part continuous stationery. For duplicate copies use 2-part NCR paper.

Troubleshooting Common Issues
❓
Bill columns are misaligned:
Check that Chrome scale is exactly 100%. Re-verify paper margins in Printing Preferences are 0.2" top, 0.3" left.
❓
Text printing too large/small:
Confirm character pitch is 10 CPI in printer properties. Font must be Courier (not proportional fonts).
❓
Paper jamming:
Re-align tractor pins. Ensure both left and right tractors are locked at correct paper width. Use proper continuous feed paper.
❓
Only half the bill prints:
Paper size in Chrome must match physical paper (13.6" × 11"). Select custom size, not A4.
❓
Printer not detected in Chrome:
Ensure Epson driver is installed. In Chrome print dialog, click "See more" to show all installed printers.
